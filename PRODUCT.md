# PinPrompt Product Context

## Product
PinPrompt is a practical AIGC learning website for creators who want to turn an idea into a usable piece of work. It combines structured courses, hands-on tutorials, tool guidance, prompt templates, favorites, and a lightweight personal learning dashboard.

## Primary Users
- AIGC beginners who do not know where to start
- Content creators and social media operators
- Designers, video creators, and brand marketers
- Existing PinPrompt users with saved prompts and favorites

## Core Outcome
Help a learner complete a first usable AI-written, AI-designed, or AI-video work through a clear sequence of learning, practice, correction, and reuse.

## Product Personality
Friendly, practical, optimistic, and creator-focused. PinPrompt should feel like an inviting creative studio with a knowledgeable teacher nearby, not an enterprise admin system or a model marketplace.

## Product Principles
1. Start with a real creative task, not abstract theory.
2. Show the next action clearly on every page.
3. Explain mistakes and how to correct them.
4. Preserve the recognizable sticky-note character without sacrificing readability.
5. Keep learning content publicly browsable; use login for saved progress, favorites, and legacy prompt data.
6. Maintain compatibility with existing local user, favorite, prompt, and API behavior.

## Anti-References
- Generic SaaS dashboards with permanent left navigation
- Dark purple AI gradients and decorative technology effects
- Dense prompt databases that do not teach a process
- Model communities, workflow canvases, enterprise admin panels, or paid membership flows

## Accessibility And Quality
- Clear text contrast and visible keyboard focus
- Reduced-motion support
- Responsive layouts for desktop and mobile
- No white screens when old or incomplete data is encountered
- Content and controls remain usable without animation
